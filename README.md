# 🧠 بنّاء العقل (Mind Builder Kids)

تطبيق ويب تفاعلي لمتابعة تطور الأبناء في بناء العقل والشخصية — حفظ القرآن، القراءة، الكتابة، وتطوير المهارات — بنظام نقاط، مستويات، أوسمة، وأهداف دورية مرنة.

## ✨ المميزات

- **تتبع متعدد الأطفال**: ملف مستقل لكل طفل بصورة رمزية خاصة
- **أقسام قابلة للتخصيص بالكامل**: عدّل، احذف، أو أضف أقسام جديدة (حفظ قرآن، قراءة، كتابة، مهارات، أو أي قسم تحدده أنت)
- **مؤشرات بصرية تحفيزية**:
  - 🧠 دماغ يمتلئ تدريجياً حسب مجموع نقاط الطفل
  - 🚗 عداد سرعة (Speedometer) بإبرة واضحة يبين التقدم نحو المستوى القادم
  - أشرطة تقدم لكل قسم على حدة
- **نظام مستويات ونقاط**: كل إنجاز يسجَّل بقوة (بسيط / جيد / ممتاز / استثنائي) مع ملاحظة اختيارية
- **أهداف دورية مرنة**: أسبوعي، شهري، أو **مخصص بتاريخ بداية ونهاية يدوي** (مناسب للإجازات)
  - تقييم تلقائي بالنجوم ⭐⭐⭐ عند انتهاء كل هدف
  - رسائل تحفيزية ذكية تتغير حسب الوقت المتبقي ومستوى النشاط
  - سجل كامل لتاريخ الأهداف السابقة
- **نظام أوسمة (Badges)**: تُفتح تلقائياً حسب الإنجاز (الانطلاقة، حافظ مجتهد، العقل الشامل، فارس المئة...)
- **عداد أيام متتالية 🔥**: لتحفيز الاستمرارية
- **حفظ تلقائي**: كل البيانات تُحفظ محلياً لكل مستخدم بدون أي إعداد إضافي

## 🚀 طريقة الاستخدام

1. افتح ملف `mind_builder.html` في أي متصفح حديث
2. أضف أول طفل بالضغط على "➕ إضافة طفل"
3. سجّل الإنجازات اليومية وتابع التقدم عبر المؤشرات البصرية
4. من ⚙ إعدادات ولي الأمر، خصّص الأقسام كما يناسب احتياجاتك
5. اضبط هدف كل طفل (أسبوعي / شهري / مخصص) من زر "⚙ تعديل الهدف"

## 🛠️ التقنيات المستخدمة

- HTML + CSS + JavaScript (Vanilla, بدون مكتبات خارجية)
- SVG للرسوم التوضيحية التفاعلية (الدماغ، عداد السرعة)
- تخزين بيانات محلي (Persistent Storage API)
